# 1. 背景（为什么）
前段时间看书，从书籍上了解到一个词汇：正向现金流。对此产生了一些好奇和疑问，我之前一直对财务的操作和思考，以及自我追求的财务自由，似乎都会有正向现金流的参与。所以，现在来梳理了解一下，什么是正向现金流？针对我个人，如何去构建我的正向现金流？

# 2. 内容（是什么）
定义：你的收入>你的支出

我们可以从3个类别来具体描述正向现金流的体现：
**经营类（主动收益，时间换钱）**
- 工作
- 副业 /  兼职
- 自由职业
- 小微生意
**投资类（被动收益，钱生钱）**
- 债股商
- 房租：住宅、公寓、商铺
- 知识付费
- 信托、REITS（我还不知道这玩意是什么）
- 保险
**融资类（杠杆收益，借力生财）**
- 低息按揭 + 高租售比
- 信用卡、消费贷免息套利
- 经营贷 / 信用贷扩大可投资产

# 3. 最佳实践（怎么做）

这里也谈不上最佳实践，因为根本就还没有实践，所以叫方案策略更合适。
**策略**
- 扩大主动收益
- 稳定增加被动收益
- 小幅探索杠杆收益
**方案框架**
- 经营类：工作、副业
- 投资类：债股商、房租、保险、知识付费
- 融资类：信用卡、低息按揭 + 高租售比
**实施方案（方案整体落入26年度计划）**
- 一阶段：1 + 2 + 1 = 工作 + 债股商 + 保险 + 信用卡
- 二阶段：2 + 3 + 1 = 工作 + 副业 + 债股商 + 保险 + 房租 + 信用卡 + 低息按揭 + 高租售比
