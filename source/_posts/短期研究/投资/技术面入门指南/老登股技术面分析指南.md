# 老登股（高股息蓝筹）标准化技术面分析指南

## 一、核心前置交易规则

1. 定位边界
    
    基本面、估值、分红决定 70% 底仓长期持有逻辑；全套技术体系仅服务 20%-30% 机动仓波段做 T，不改变长线底仓持有决策。
2. 周期研判优先级
    
    月线定长期牛熊→周线为波段 T 唯一核心决策周期→日线仅辅助精准买卖价位，禁止单独依靠日线开仓、减仓。
3. 统一量化判定标准

- 有效长上 / 下影线：影线长度 ÷K 线实体幅度≥2 倍，低于该比值视为盘中杂波，无参考价值；
- 地量：单周成交量≤5 周均量 50%；放量分歧：单周成交量≥5 周均量 2 倍；
- 有效站稳 250 日年线：连续 3 周收盘价位于年线上方；有效跌破年线：连续 2 周收于年线下 + 周度阴跌放量；

4. 仓位硬性约束
    
    常态 70% 底仓长期不动，仅动用 20%-30% 机动仓波段操作；个股估值近 5 年分位＜10% 单边上涨行情，停止一切做 T 操作。

## 二、标准化分步分析流程

### 第一步：月线长期趋势研判（判定底仓安全性）

1. 250 年线运行状态：股价是否连续 3 个月站稳年线，均线呈多头 / 震荡 / 空头排列；
2. 月 K 形态与月级别量能：有无达标长下影 / 长上影，近三月量能为缩量震荡、上涨温和放量、阴跌持续放量；
3. 月线 MACD：零轴上方多头 / 零轴下方空头，是否存在大级别顶背离、底背离；
4. 月线综合定性：长期多头 / 中性震荡 / 长期走熊。

### 第二步：周线核心波段信号研判（机动仓操作核心依据）

1. 均线系统（5/20/60/250 周线）
    
    标注当前多重支撑位、压力位；判断短中长期均线多头 / 空头排列。
2. 量价匹配关系
    
    判定本周量能等级：地量 / 温和放量 / 巨量分歧 / 阴跌放量；区分健康量价（上涨放量、下跌缩量）、风险量价（上涨缩量、下跌放量）。
3. 周 K 线形态
    
    确认本周 K 线：达标长下影、达标长上影、普通震荡 K 线。
4. 四大核心指标周线信号
    
    （1）MACD：金叉 / 死叉，是否形成顶背离、底背离，DIF 位于零轴上方 / 下方；
    
    （2）BOLL 布林带：股价触碰下轨 / 中轨 / 上轨，轨道收口观望 / 开口单边行情；
    
    （3）RSI 相对强弱：数值＜30 超卖、30-70 中性区间、＞70 超买；
    
    （4）筹码分布：低位筹码单峰密集锁仓、低位筹码持续上移出货、多峰分散。
5. 走势结构与共振位
    
    走势分类：箱体震荡 / 单边慢牛 / 下降通道；支撑、压力点位重合 2 个及以上判定为强有效共振位。

### 第三步：日线辅助验证（仅微调买卖点位，不独立决策）

1. 日线价格贴合周线支撑、压力区间情况；
2. 日线短期量能、小级别指标辅助甄别周线信号真伪；
3. 机动仓止损基准：买入后跌破支撑位且放量下跌 3% 为止损线。

### 第四步：基本面一票否决校验（优先级高于全部技术信号）

出现任意一条，放弃加仓 / 新建 T 仓，优先减仓收缩机动仓：

1. 连续年度分红下滑、股息率持续走低；
2. 企业经营性现金流长期恶化；
3. 行业出台重大压制政策、长期基本面利空；
4. 估值处于近 5 年 80% 以上高位区间。

## 三、当日操作结论判定标准（四选一）

### 1. 适合建仓 / 加仓机动 T 仓（需同时满足≥2 项共振条件）

1. 月线长期多头，股价稳定站于年线上方；
2. 周线回踩 60 周线 / 箱体下沿 / 布林下轨多重支撑；
3. 周成交量达到阶段地量标准；
4. 周线 MACD 底背离、周 RSI 数值低于 30 超卖区间；
5. 周线收达标长下影，下方承接力度充足；
6. 基本面无利空，估值处于历史低位区间。
    
    操作执行：机动仓分两笔分批低吸，常规波段止盈 5%-8%，跌破支撑 3% 无条件止损。

### 2. 减仓 / 清仓全部机动 T 仓（触发任意 1 条即可执行）

1. 周线价格触及箱体上轨、布林带上轨压力；
2. 周 RSI＞70 超买区间、周线 MACD 形成顶背离；
3. 连续上涨后周线巨量分歧 + 收达标长上影；
4. 股价有效跌破 250 年线，基本面无改善预期；
5. 个股估值进入历史高位区间。

### 3. 观望，不进行任何 T 仓操作

1. 布林轨道持续收口，市场波动收窄，等待方向选择；
2. 周线全部指标处于 30-70 中性区间，无背离、无超买超卖信号；
3. 股价运行在支撑、压力中间区间，无多重共振信号；
4. 极致低估单边慢牛行情，规避做 T 卖飞主升段。

### 4. 禁止任何加仓操作，只减不加

1. 月线有效跌破年线，长期趋势走熊；
2. 个股处于标准下降通道，阶段高点持续下移；
3. 基本面出现分红、现金流、行业利空预警。

## 四、配套实操避坑规则

1. 单一 K 线、单一指标信号不具备操作价值，必须多重信号共振再交易；
2. 下降通道、有效跌破年线标的，禁止抄底、分批摊薄成本；
3. 老登股杜绝高频日内 T，仅依托周线开展波段操作，减少手续费损耗；
4. 指标无需叠加十余个，均线 + 量价 + MACD + 布林带 + 筹码分布足够研判；
5. 所有影线、量能、突破跌破判定严格执行量化标准，杜绝主观感受判断。
